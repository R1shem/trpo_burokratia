# trpo_burokratia
Учебный проект.

WPF-приложение, написанное в Visual Studio на C#. Используются Json-файлы для записи и чтения данных (при тестировании возможны проблемы, связанные с пакетом NuGet).

Функционал: авторизация, просмотр, добавление, редактирование, удаление информации о сотрудниках, клиентах, возможно операциях. Вывод операций в Excel.

Было выполнено в начале-середине 2023 года, во время обучения на втором курсе.
